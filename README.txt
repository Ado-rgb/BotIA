📦 BOT IA - Adonix 🤖

1. Asegúrate de tener instalado Node.js v18 o superior.
2. Abre una terminal en esta carpeta y escribe:

   npm install

3. Luego ejecutá el bot con:

   npm start

4. Escanea el código QR con WhatsApp y ¡listo!
5. Escribile cualquier mensaje y responderá con la IA Adonix.
